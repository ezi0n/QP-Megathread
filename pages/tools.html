Standalone Tools (PC Sideloaders)

These PC applications sideload Quest content to your headset. They do not include a built-in server — you'll need to configure one separately.

  • Rookie (Official): The original sideloading tool. Needs a server to function — see the Rookies JSON page for setup info. Download from GitHub
  • ApprenticeVR (Best Fork): The best fork of Rookie with improvements. Also needs a server — see the Rookies JSON page for setup info. ApprenticeVR on GitHub
  • QLoader: An alternative sideloader. Needs some work to add a server. QLoader on GitHub

Note: Rookie and ApprenticeVR both need a server configured. The easiest way is via the Rookies JSON page. For a full walkthrough including the server, see Rookie 3.0.1 Setup Guide.

Web Tools (WebUSB Sideloaders)

These websites allow you to connect to your headset and sideload apps. You’ll need a browser that supports WebUSB (such as Google Chrome, Microsoft Edge, Opera, or Brave).

  • Quest Webloader: Provided by the group "QuestArchive". Quest Webloader Website
  • QuestLoad: Provided by Frizzle. QuestLoad Website

Note: These websites won’t work in browsers like Mozilla Firefox or Safari, which don’t support WebUSB. Make sure you use Google Chrome, Microsoft Edge, Opera, or Brave.
