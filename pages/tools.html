🛠️ Standalone Tools

Community-made standalone tools for Quest users, including PC sideloaders, browser-based loaders, and other useful utilities.

💻 PC Sideloaders

These PC applications sideload Quest content to your headset. They do not include a built-in server — you must configure one separately. For sideloader with servers, see Quest Standalone Sources.

  • Rookie (Official): The original sideloading tool. Requires a server. See the public JSON page for setup. Download
  • ApprenticeVR (Best Fork): Improved fork of ApprenticeVR. Also requires a server. See the public JSON page. ApprenticeVR on GitHub
  • QLoader: Alternative sideloader. Requires manual server setup. GitHub
  • LFI Sideloader: Alternative sideloader. Lightweight APK and OBB sideloader. GitHub
  • Quest File Transfer (QFT): Alternative sideloader for manual file transfer using a dual-pane file manager. GitHub

Note: Rookie and ApprenticeVR require a server configuration. The easiest method is via the public JSON. For a full walkthrough, see Rookie 3.1 Setup Guide .

🌐 WebLoaders

These browser-based tools allow direct sideloading or Quest management through WebUSB. Requires a supported browser such as Chrome, Edge, Opera, or Brave.

  • Quest Webloader: Browser-based sideloading tool for Quest. Website
  • QuestLoad: Simple browser-based sideloading utility. Website
  • Web ADB Menu: A web-based ADB tool for Meta Quest. Does much more than sideloading, including access to various Quest performance settings. Website

Note: These will not work in Firefox or Safari because those browsers do not support WebUSB.

🧩 General Standalone Tools

Other standalone utilities that do not fit directly into the sideloader categories.

  • APC (Automagic Package Changer): A desktop utility that renames Quest APK package names for easier app management and conflict avoidance. It handles the package-changing process automatically with a simple drag-and-drop workflow. GitHub
