🛠️ Standalone Tools (PC Sideloaders)

These PC applications sideload Quest content to your headset. They do not include a built-in server — you must configure one separately.

  • Rookie (Official): The original sideloading tool. Requires a server. See the public JSON page for setup. Download
  • ApprenticeVR (Best Fork): Improved fork of ApprentiveVR. Also requires a server. See the public JSON page. ApprenticeVR on GitHub
  • QLoader: Alternative sideloader. Requires manual server setup. GitHub
  • LFI Sideloader: Alternative sideloader. Lightweight apk and obb sideloader. GitHub
  • Quest File Transfer (QFT): Alternative sideloader: Manual File Transfer using a dual-panes file manager GitHub

Note: Rookie and ApprenticeVR require a server configuration. The easiest method is via the public JSON. For a full walkthrough, see Rookie 3.1 Setup Guide .

🌐 Web Tools (WebUSB Sideloaders)

These sites allow direct sideloading via browser. Requires WebUSB support (Chrome, Edge, Opera, Brave).
