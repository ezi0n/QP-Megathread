🧰 Community Tools

Split into Standalone, PCVR and General — built and maintained by the community.

Standalone PCVR General
🛠️ Standalone Tools These PC applications sideload Quest content to your headset.
They do not include a built-in server — you'll need to configure one separately.
  • Rookie (Official): The original sideloading tool. Needs a server to function — see the Rookies JSON page for setup info. Download from GitHub
  • ApprenticeVR (Best Fork): The best fork of Rookie with improvements. Also needs a server — see the Rookies JSON page for setup info. ApprenticeVR on GitHub
  • MetaFather: All-in-one headset manager and sideloader with its own server for downloading games. MetaFather

Note: Rookie and ApprenticeVR both need a server configured. The easiest way is via the Rookies JSON page. For a full walkthrough including the server, see Rookie 3.0.1 Setup Guide.

🧩 General Tools Cross-platform or generally useful tools go here.
  • Quest Webloader: WebUSB Sideloader provided by QuestArchive. Quest Webloader Website
  • QuestLoad: WebUSB Sideloader provided by Frizzle. QuestLoad Website

Note: These websites won’t work in browsers like Mozilla Firefox or Safari, which don’t support WebUSB. Make sure you use Google Chrome, Microsoft Edge, Opera, or Brave.

🖥️ PCVR Tools Tools to enhance, configure, and troubleshoot your PC VR experience.
  • ARMGDDN: A massive community-driven resource hub specializing in high-speed access to PC and PCVR games. ARMGDDN Official Website

Note: For more PC VR content, please check our PCVR Sources page.
