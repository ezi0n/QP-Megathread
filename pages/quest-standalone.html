🥽 Quest Standalone Sources

Curated links, mirrors, and libraries for native Quest content, with badges to help you quickly see which sources use desktop sideloaders and which are more flexible.

Badge Key Non-Telegram Sources Telegram Sources

Badge Key

  • Desktop Sideloaders
    These are sideloaders you run on your desktop PC, usually Windows, though some are multi-OS. They handle downloading and sideloading for you, and usually come with a gamelist included or can have one added easily.
  • Flexible
    These are download-only sources and require a method to sideload. Depending on your preference and use case, that can be done natively on the Quest, with a PC sideloader, on Android, or anything in between.

In general, desktop sideloaders are often the most convenient option for people at home with a computer. Flexible sources are useful if you are traveling, do not have a PC, or just prefer not to use one.

Telegram sources follow the same idea. They are also flexible and can be used with or without a desktop application depending on your workflow and preference.

Non-Telegram Sources

These sources do not require Telegram. The first four are desktop sideloaders. Everything after that is flexible and download-only.

  • Rookie's Sideloader: Desktop Sideloaders
    The official, iconic sideloader from the Legendary VRPirates. Rookie's Sideloader. Since it no longer comes with the VRP gamelist out of the box, RSL has been updated to allow the public.json method or an rclone config.
  • Rookie's - The Easy Way: Desktop Sideloaders
    Full walkthrough including server, designed to get you up and running fast. Rookie 3.2 Setup Guide
  • VRSideForge: Desktop Sideloaders
    A cool torrent scraper for Rutracker with a built-in sideloader. Great project for finding and installing content in one workflow. VRSideForge on GitHub
  • MetaFather: Desktop Sideloaders
    A site with a sideloader that includes games. It seems to have a bit of an unsavory reputation, but not really sure why. Seems safe. MetaFather.org
  • Rutracker: Flexible
    Large selection of Quest content available via torrents. Can be hard to cut through the chaff, and torrents are somewhat limiting for this use case, but some people prefer it. RUTracker
  • VRtor: Flexible
    A Russian website with a vast collection of Quest games that are updated quite often. It offers multiple download sources, including direct links from file hosting sites or Telegram. VRtor
  • YAOSVR: Flexible
    Focuses on older stuff but may be attractive to those without PCs who need direct access to content. YaosVR
  • BaozouVR: Flexible
    Basic Chinese site, similar to VRMOO.net but free. BaozouVR

Note: This is not an exhaustive list. It is a starting point. If you know of other non-Telegram sources, open a PR to add them.

Telegram Sources

These sources operate through Telegram channels and groups. All of them except for MetaQuestNexus are flexible and can be used with or without a desktop application depending on your use case and preference.

  • MetaQuestNexus: Desktop Sideloaders
    A small Spanish-language group that uses QLoader. Does not have a huge library, but has some stuff not available elsewhere. t.me/MetaQuestNexus
  • the_vrSrc: Flexible
    Provides info that works with Rookie and ApprenticeVR. Other tools can be adapted to work with it as well. Currently the best source available. t.me/the_vrSrc
  • VRlite: Flexible
    The Telegram group linked to the VRtor website from above. Everything is in Russian, but it offers many games available for direct download through Telegram. t.me/VRlite
  • NSWTL Gamehub: Flexible
    A bot-managed multi-platform torrent game search and download source, seeders permitting. Claimed to be the world's largest database. Select group VR Oculus Quest. t.me/NSW_TorrentLibrary
  • CosMos VR: Flexible
    Yet another Chinese source for torrents. You can just search for a game in English. t.me/CosMos_VR

Note: This is not an exhaustive list. If you know of other Telegram sources, open a PR to add them.
