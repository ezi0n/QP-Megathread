🥽 Quest Standalone Sources

Curated links, mirrors and libraries for native Quest content — no PC required.

Non-Telegram Sources

These sources don't require Telegram and can be accessed directly via the web or desktop apps.

  • Rookie's Sideloader: The official, iconic sideloader from the Legendary VRPirates. Rookie's Sideloader. Since it no longer comes with the VRP gamelist out of the box, RSL has been updated to allow the public.json method or an rclone config.
  • Rookie's — The Easy Way: Full walkthrough including server, designed to get you up and running fast. Rookie 3.2 Setup Guide
  • VRSideForge: A cool torrent scraper for Rutracker with a built-in sideloader. Great project for finding and installing content in one workflow. VRSideForge on GitHub
  • Rutracker: Large selection of Quest content available via torrents. Can be hard to cut through the chaff, and torrents are somewhat limiting for this use case, but some people prefer it. RUTracker
  • VRtor: A Russian website with a vast collection of Quest games that are updated quite often. It offers multiple download sources, including direct links from file hosting sites or Telegram. VRtor
  • MetaFather: A site with a sideloader that includes games, it seems to have a bit of an unsavory reputation but not really sure why. Seems safe. MetaFather.org
  • YAOSVR: Focuses on older stuff but may be attractive to those without PCs who need direct access to content. YaosVR
  • BaozouVR: Basic chinese site, similar to VRMOO.net but FREE. BaozouVR

Note: This is not an exhaustive list — it's a starting point. If you know of other non-Telegram sources, open a PR to add them.

Telegram Sources

These sources operate through Telegram channels and groups. You'll need the Telegram app to access them.

  • the_vrSrc: Provides info that works with Rookie and ApprenticeVR. Other tools can be adapted to work with it as well. Currently the best source available. t.me/the_vrSrc
  • VRlite: The Telegram group linked to the VRtor website from above. Everything is in Russian, but it offers many games available for direct download through Telegram. t.me/VRlite
  • MetaQuestNexus: A small Spanish-language group that uses QLoader. Doesn't have a huge library, but has some stuff not available elsewhere. t.me/MetaQuestNexus
  • NSWTL Gamehub: A bot managed multi-platform torrent game search and download. Seeders permitting. Claimed to be world's largest database (sic). Select group VR Oculus Quest. t.me/NSW_TorrentLibrary

Note: This is not an exhaustive list. If you know of other Telegram sources, open a PR to add them.
